# CashGuard - 电竞风个人记账APP

## 项目定位
CashGuard（钱盾 / 现金卫士）是一款融合电竞元素的个人记账与资产管理APP，将预算比作“血量护盾”，将存钱视为“守护财富”，让理财过程更具游戏化体验。

## 品牌信息
- **官方中文名**：钱盾 / 现金卫士
- **Slogan**：
  - 英文：Your money, your shield. / Guard your cash, win with wealth.
  - 中文：你的财富，你的护盾。 / 守护每一分，理财赢到底。

## 视觉设计规范
- **LOGO 方向**：盾牌 + 钱币 + 科技线条
- **风格**：电竞、赛博朋克、锐利、未来感
- **主色**：霓虹蓝 #13E2F5 + 深黑 #12141D
- **字体**：CashGuard 使用粗体电竞字体，搭配光效

## 核心功能
- 预算管理（血量护盾）
- 收支记账
- 财富目标（守护财富）
- 游戏化激励机制
